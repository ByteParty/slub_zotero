Known Problems
==============

At the moment the plugin is made for a specific way of naming the bibliography.
In detail the sub collections have to have the pattern "\*year" e.g. "first sub collection 2019".
The last four characters of the name string are extracted to be used as "headline".
Internally this sub string is also used as index of an array.
