Users Manual
============

The following document holds information about the installation of the TYPO3
SLUB Zotero extension. In case of further questions please contact the
release management.

This TYPO3 extension is primarily meant to visualize a zotero.org hosted
bibliography on a TYPO3 website.

`zotero.org <https://www.zotero.org>`_ is a free and open-source reference
management software to manage bibliographic data and related research materials. There is a
registration necessary but is free to a certain point. To use this plugin
you need some informations like user or group identifiers and maybe an authorization token to get the data via API.
